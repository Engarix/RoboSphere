# Controlller

## Controls:
### Basic Movement
* UP - Forward
* DOWN - Backwards
* LEFT - Rotate left
* RIGHT - Rotate right

### Advanced Movement
* UP + LEFT - Forward with left turn 45 degrees
* UP + RIGHT - Forward with right turn 45 degrees
* DOWN + LEFT - Backwards with left turn 45 degrees
* DOWN + RIGHT - Backwards with right turn 45 degrees

## Commands:
### MOVE
MOVE 1 0 : Forward

MOVE -1 0 : Backward

MOVE 0 -1 : Rotate Left

MOVE 0 1 : Rotate Right

...
